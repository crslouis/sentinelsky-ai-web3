# SentinelSky AI Agent Demo

This is a demonstration of the SentinelSky AI intelligence layer using **MOCK DATA ONLY**.

## ⚠️ Important Notice

- **No real credentials** are used
- **No real wallets** are connected
- **No real transactions** are executed
- All data is simulated for demonstration purposes

## 🚀 Running the Demo

```bash
python3 main.py
```

## 📋 What the Demo Shows

1. **Opportunity Scanning**: Simulates finding Web3 opportunities
2. **Risk Analysis**: Demonstrates intelligent analysis of opportunities
3. **Workflow Planning**: Shows how actions would be prepared
4. **Safety Checks**: Highlights human-in-the-loop approval requirements

## 🛡️ Safety Features Demonstrated

- Human approval required for all transactions
- No private key storage or handling
- Transparent action logging
- Risk assessment before execution

## 🔧 Requirements

- Python 3.8+
- No external dependencies (uses stdlib only)

## 📝 Output

The demo produces:
- List of mock opportunities
- Detailed analysis of selected opportunity
- Simulated workflow with safety checks
- Clear indication that all data is mock
